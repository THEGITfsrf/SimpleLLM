# Zombie Game 🧟

A simple zombie survival game built with Python.

## Features
- Player health system
- Zombie waves with increasing difficulty
- Score tracking
- Dynamic round progression
- Interactive gameplay

## How to Run

```bash
cd zombie-game
python main.py
```

## Controls
- Type `y` to play another round
- Type `n` to end the game
- Game continues automatically until you choose to stop

## Game Rules
- Survive waves of zombies
- Each round increases difficulty (more zombies)
- Maintain your health to continue playing
- Score points by surviving rounds

- `main.py` - Main game application
